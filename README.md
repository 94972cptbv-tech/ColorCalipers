# Color Calipers

Color Calipers is a color harmony and color sampling app for iPhone and iPad.

Use live color calipers, photo sampling, harmony rules, saved palettes, and paint-card references to build better palettes for rooms, clothing, furniture, artwork, branding, and other creative projects.
